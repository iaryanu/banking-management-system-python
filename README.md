# Banking Management System (Python)

A command-line banking application that simulates real-world banking operations.

---

## Features
- Account creation using PAN and Date of Birth
- Automatic unique account number generation
- Secure login authentication
- Deposit and withdrawal operations
- Balance validation (prevents overdraft)
- Fund transfer between accounts
- Password change functionality
- Transaction history tracking

---

## Security Features
- Password validation
- Login attempt limit (3 tries)
- Admin protected account view

---

## Concepts Used
- Object Oriented Programming
- Modular programming (multiple files)
- Input validation
- Date handling
- List data structures
- Class design and encapsulation
